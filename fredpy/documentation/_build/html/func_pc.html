fredpy.series.pc¶

series.pc(log=True, method='backward', annualized=False)¶

Computes the percentage change in the data from the previous period.

Parameters:
  • log (bool) – If True, computes the percentage change as \(100\cdot\log(x_{t}/x_{t-1})\). If False, compute the percentage change as \(100\cdot\left( x_{t}/x_{t-1} - 1\right)\).
  • method (string) – If ‘backward’, compute percentage change from the previous period. If ‘forward’, compute percentage change from current to subsequent period.
  • annualized (bool) – If True, percentage change is annualized by multipying the simple percentage change by the number of data observations per year. E.g., if the data are monthly, then the annualized percentage change is \(4\cdot 100\cdot\log(x_{t}/x_{t-1})\).
Returns:

series
